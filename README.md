# Blog_Notas_Minimalista
## Blog Minimalista
### Descripción
### El proyecto consiste en un blog donde los usuarios pueden registrarse, iniciar sesión, crear publicaciones y comentar en las publicaciones de otros usuarios.
### Problema que resuelve
### Permite compartir ideas y contenido de manera sencilla, además de que los usuarios pueden interactuar mediante comentarios y recibir notificaciones.
